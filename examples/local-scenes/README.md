# Local Scene Templates

These templates are intentionally generic.

They are meant to show:

- a branched asset pipeline
- an independent long-form flow

You are expected to adapt labels, colors, and node descriptions to your own project.

Workflow:

1. copy one template
2. edit the local scene file
3. preview with local Excalidraw MCP
4. if you need browser upload/open, run `node scripts/export_native_scene.js your_scene.elements.json`
5. export a share link only if needed
