# Upstream Attribution

This repo is an integration layer for the upstream Excalidraw MCP project.

Upstream:

- `excalidraw/excalidraw-mcp`
- <https://github.com/excalidraw/excalidraw-mcp>

This repo intentionally does not rebrand the upstream server as a separate original implementation.

What this repo adds:

- local-first skill packaging
- local scene workflow guidance
- reusable scene templates
- online Claude browser playbooks
- missing-text share-link troubleshooting
